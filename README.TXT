80-103A USERS......


NOTE THAT NONE OF THE 3 PROGRAMS (REMOTE, INSTALL, MM100) WILL
RUN WITH A 80-103A UNTIL YOU MODIFY THE 50 MS TIMING LOOP.
THEY NOW USE THE MICROMODEM 100 HARDWARE TIMER.  YOU CAN
CHANGE THEM TO A SOFTWARE 50 MS DELAY.  THE DELAY SUBROUTINE
 IN THE 80-103A MANUAL SHOULD BE USABLE.
